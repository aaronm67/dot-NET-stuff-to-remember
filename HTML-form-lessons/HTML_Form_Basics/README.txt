This project demonstrates the following things:
 - What a regular postback looks like in MVC
 - How to do AJAX postback and partial updates.
	- You can do this with the out of the box .NET Ajax stuff 
	- You can do this with jAuery
 - How to add to a .NET object collection on the client side and have it parse correctly on post back
 - How to set up an editor template (these save some time!).
